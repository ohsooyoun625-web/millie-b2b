millie-b2b_handoff.zip

첨부 파일로 전달된 프로젝트 패키지

첨부

Node.js LTS

nodejs.org에서 LTS 버전 다운로드 후 설치

설치 필요
1

Claude Code 설치

claude.ai/code 접속 → 다운로드 후 설치

2

프로젝트 압축 해제

받은 zip 파일을 원하는 위치에 압축 해제

3

패키지 설치

압축 푼 폴더 안에서 터미널을 열고 아래 명령어 입력

npm install

💡 터미널 여는 법: 폴더 안 빈 곳에서 Shift + 우클릭 → "여기서 PowerShell 창 열기"

4

Claude Code로 폴더 열기

Claude Code 실행 → Open Folder → 압축 푼 폴더 선택

5

개발 서버 실행

Claude Code 하단 터미널에서 입력:

npm run dev

💡 브라우저에서 http://localhost:3000 접속하면 페이지 확인 가능

6

Claude에게 작업 이어달라고 요청

Claude Code 채팅창에 이렇게 입력하면 됩니다:

“이전 작업 이어서 해줘”

💡 CLAUDE.md 파일이 있어서 Claude가 프로젝트 맥락을 자동으로 파악합니다.

문제가 생기면 연락주세요!
